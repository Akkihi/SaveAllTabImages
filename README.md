# Save All Tab Images

Расширение для Google Chrome, которое собирает ссылки на изображения из всех вкладок текущего окна и скачивает их в выбранную подпапку загрузок.
Поддерживает `img`, `srcset` и фоновые `background-image`, сохраняет файлы как есть (включая `.webp`, `.jpg`, `.png` и т.д.).
После первой загрузки открывает папку с файлами.

## Что умеет
- «Download all images» — сохранить все найденные изображения из всех вкладок окна (дубликаты по URL пропускаются).
- «Download largest image from each tab» — выбрать одно самое крупное изображение с каждой вкладки и скачать.
- «Change folder» — указать подпапку внутри стандартной папки загрузок; выбор сохраняется в `chrome.storage`.
- После завершения первой загрузки открывает проводник в папке с файлом.

## Установка
1. Соберите файлы из репозитория в папку (папка должна содержать `manifest.json`, `background.js`, `popup.html`, `popup.js`, каталог `icons`).
2. Откройте `chrome://extensions`, включите режим разработчика.
3. Нажмите «Загрузить распакованное» и выберите папку проекта.

## Использование
1. Кликните по иконке расширения — откроется popup.
2. При первом запуске укажите подпапку внутри загрузок (по умолчанию `AllTabsImages`); папка создастся автоматически.
3. Выберите действие:
   - «Download all images» — скачать все.
   - «Download largest image from each tab» — по одному крупному на вкладку.
   - «Change folder» — сменить подпапку.
4. По завершении загрузки откроется папка с первым скачанным файлом.

## Ограничения и заметки
- Работает только на вкладках `http/https/file`; служебные страницы (`chrome://`, `edge://` и т.п.) пропускаются.
- API Chrome не позволяет выбирать произвольный путь вне папки загрузок; можно указать только подпапку.
- Если в Chrome включен запрос пути на каждый файл, диалоги сохранятся (это системная настройка).
